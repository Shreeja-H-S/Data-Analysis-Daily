# Day 2 - Titanic Dataset Analysis

## Objective
Analyze the Titanic dataset to understand factors affecting passenger survival.

## Tools
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Explored the dataset
- Checked missing values
- Calculated survival rates
- Compared survival by gender and passenger class
- Visualized key findings

## Key Insights
- Female passengers had a higher survival rate.
- First-class passengers survived more frequently.
- Missing values were found in Age and Cabin.
