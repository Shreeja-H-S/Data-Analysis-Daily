# Day 1 - Sales Data Analysis

## Objective
Analyze sales data to identify top-performing product lines and customers.

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
- Loaded the dataset
- Explored the data
- Calculated total sales by product line
- Identified top customers
- Analyzed sales by country
- Visualized sales using bar and pie charts

## Key Insights
- Classic Cars generated the highest total sales.
- A few customers contributed significantly to overall revenue.
- Sales were concentrated in a few countries.
